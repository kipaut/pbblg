# Объекты

## Содержание
- [\<user>](#user)
- [\<game>](#game)

### user 
```json
{
    "userId": 1,
    "name": "Sebas"
}
```
 
 ### game 
```json
{
    "gameId": 123,
    "userCount": 2,
    "status": 1,
    "created": "2018-05-28 08:39:44"
}
```

Сатусы могут быть: OPEN(1), IN_PROGRESS(2), ENDED(3).
